# Aplicación de Reserva de Videojuegos

## Descripción
Proyecto de diseño y documentación de una aplicación de reserva de videojuegos.

La aplicación permite a los usuarios:
- Explorar un catálogo de videojuegos.  
- Gestionar reservas activas.  
- Crear y consultar listas de deseos.  
- Controlar compras y descargas de juegos adquiridos.  
- Gestionar perfil de usuario.
